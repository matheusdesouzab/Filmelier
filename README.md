## Baixar projeto

```
git clone https://github.com/matheusdesouzab/Projeto-Formulario.git
```

## Baixa Dependências
```
npm install
```

### Inicializar servidor
```
npm run serve
```

Se tudo ocorrer bem, a aplicação estara disponível em [localhost:8080](http://localhost:8080)
